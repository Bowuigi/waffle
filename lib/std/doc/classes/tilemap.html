Methods

tilemap:new (w, h, fill)
Create a new tilemap
Do not call this function directly, use tilemap(w,h,fill) instead.

Parameters:

  • w number Width of the tilemap
  • h number Height of the tilemap
  • fill Fill the tilemap with this value
tilemap:copy ()
Copy a tilemap to another one.
Be aware that this creates a new tilemap

Returns:

    tilemap Copy of the tilemap
tilemap:tileExists (x, y)
Check if a tile exists on the given tilemap

Parameters:

  • x number X position of the tile
  • y number Y position of the tile

Returns:

    boolean True if the tile exists, false otherwise
tilemap:tileOnBounds (x, y)
Check if a position is inside the tilemap

Parameters:

  • x number X position of the tile
  • y number Y position of the tile

Returns:

    boolean True if the tile is inside of the tilemap bounds, false otherwise.
tilemap:getTile (x, y)
Get a certain tile of the tilemap
If no tile is found at the position, nil is returned and a warning is emitted

Parameters:

  • x number X position to get the tile from
  • y number Y position to get the tile from

Returns:

    Tile obtained or nil if no tile was found
tilemap:setTile (x, y, tile)
Set a tile on the tilemap
If the tile is not on bounds then false is returned and a warning is emitted.

Parameters:

  • x number X position where the tile goes
  • y number Y position where the tile goes
  • tile Tile inserted at the position

Returns:

    boolean True if the tile could be set, false otherwise
tilemap:fillTiles (startX, startY, endX, endY, fill)
Fill the tiles on a tilemap (make a rectangle) with the given value
If any of the positions in between those values is out of bounds, a warning is emitted.

Parameters:

  • startX number X position to start with
  • startY number Y position to start with
  • endX number X position to go to
  • endY number Y position to go to
  • fill Tile to fill with (can have any type)
