init (settings)
Initializes the logging library with the given values (or the default ones if those are nil)

Parameters:

  • settings table Settings for the log functions

Usage:

  • log.init()
  • log.init({logFile=fileHandler})
  • log.init(log.settings)
error (context, message)
Outputs an error to the log file and exits with status code 1

Parameters:

  • context string Context of the error message
  • message string Error message

Usage:

  • log.error("Waffle", "this is a fatal error!")
  • log.error("Logger", "this is another fatal error!")
warning (context, message)
Outputs a warning to the log file

Tables

settings
Log settings used to configure the logging functions

Fields:

  • dieOnWarning If this is enabled, the program exits with code 1 if a warning is encountered
  • printWarnings If this is true, warnings will be printed to the log file
  • logFile File handler to the file to print logs to, standard error by default
