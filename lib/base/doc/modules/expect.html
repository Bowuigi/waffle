Module expect

Expect module

Methods
type (expected, got)
Check for type, can also be called with expect(expected, got)
On error, the function doesn't return, but it emits an error as if it was a normal lua error. Otherwise, it continues execution normally

Parameters:

  • expected string/table Expected type, or table containing any of the expected types
  • got Type received, must be a string and the result of the type() function

Usage:

    -- This expects a string, but gets a number, so it throws an error
expect("string", type(1))
-- This expects either a string or a table, but gets a function, so it throws an error
expect({"string","table"},type(print))
length (expected, got)
Check for length
On error, the function doesn't return, but it emits an error as if it was a normal lua error. Otherwise, it continues execution normally

Parameters:

  • expected Expected length, must be a number, otherwise, an error is thrown
  • got Length received, must be a number, otherwise, an error is thrown

Usage:

    expect.length(3,#("hello")) -- Expect an object of length 3, but gets an object of length 5, so it throws an error
