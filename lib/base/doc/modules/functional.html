Module functional

A collection of functions from functional programming

Functions

head (list) Get the first element of a list (doesn't make much sense in Lua, but it is here anyway)
tail (list) Get every element other than the first and build a table with it
Note: It will only work with index k,v pairs, like {1,2,3,4}, but not on tables like {bob = 1, anna = 2}
last (list) Get the last element of a list (again this doesn't make much sense in Lua, but here it is)
compose (func1, func2) Create a function that runs func1(func2(param)) when called with a parameter
expand (param, times, func) Expand a function into multiple ones and execute them
fold (sequence, initial, func) Fold a list, similar to expand
foldr (sequence, initial, func) Perform a fold, but in reverse
reduce (list, func) Reduce a list, equivalent to list:fold(head(list), tail(list))
The first value of the list acts as the initial value, the rest is the table to operate on
zip (list1, list2) Combine both lists into one of the form {{list[1], list2[1]}, {list[2], list2[2]}...}
filter (list, func) Filter a list by passing it into a function
each (list, func) Run a function for each element on the list without storing the result
eachi (list, func) Run a function for each integer key element on the list without storing the result
eachk (list, func) Run a function for each element on the list without storing the result
eachki (list, func) Run a function for each integer key element on the list without storing the result

compose (func1, func2)
Create a function that runs func1(func2(param)) when called with a parameter

Parameters:

  • func1 function First function in the nested call
  • func2 function Second function in the nested call

Returns:

    function Function returning a nested call in form of function(x) return func1(func2(x)) end

Usage:

    compose(math.sin, math.abs)(-5) -- Get the sin of the absolute of -5, AKA math.sin(math.abs(-5)) or math.sin(5)
expand (param, times, func)
Expand a function into multiple ones and execute them

Parameters:

  • param Parameter to pass to the functions
  • times number How much times to nest the function
  • func function Function to use on expansion

Returns:

    Result of the expansion

Usage:

    -- This is how you would get an index of a linked list, but on Lua (this shouldn't be done, it is just an example)
expand({1,2,3,4,5}, 3, tail) -- tail(tail(tail({1,2,3,4,5}))) or in it's more readable form, ({1,2,3,4,5}):tail():tail():tail() which resembles UNIX shell pipes
-- This results in {4,5}, so we just apply head to it
head(expand({1,2,3,4,5}, 3, tail)) -- This gives us 4
fold (sequence, initial, func)
Fold a list, similar to expand

Parameters:

  • sequence table List of items to use as secondary parameters sequentially
  • initial Initial value for the first argument
  • func function Function to execute

Returns:

    Result of the expansion

Usage:

    ({1,2,3}):fold(0, sum) -- 0 | sum 1 | sum 2 | sum 3 // sum(sum(sum(0,1),2),3)
foldr (sequence, initial, func)
Perform a fold, but in reverse

Parameters:

  • sequence table List of items to use as primary parameters sequentially (but in reverse)
  • initial Initial value for the second argument
  • func function Function to execute

Returns:

    Result of the expansion

Usage:

    ({1,2,3}):foldr(4,sum) -- sum(1,sum(2,sum(3,4)))
reduce (list, func)
Reduce a list, equivalent to list:fold(head(list), tail(list))
The first value of the list acts as the initial value, the rest is the table to operate on

Parameters:

  • list table List to reduce
  • func function Function to apply

Returns:

    Result of the expansion

Usage:

    ({1,2,3}):reduce(sum) -- fold(1,{2,3}) // sum(sum(1,2),3)
zip (list1, list2)
Combine both lists into one of the form {{list[1], list2[1]}, {list[2], list2[2]}...}

Parameters:

  • list1 table First list, takes the first place on the resulting list
  • list2 table Second list, takes the second place on the resulting list

Returns:

    table List on the form of {{list[1], list2[1]}, {list[2], list2[2]}...}

Usage:

    zip({1,2,3}, {4,5,6}) -- {{1,4},{2,5},{3,6}}
filter (list, func)
Filter a list by passing it into a function

Parameters:

  • list table List to filter

Returns:

    table Filtered list

Usage:

    ({1,2,"e",3,"hi",{}}):filter(isnumber) -- {1,2,3}
